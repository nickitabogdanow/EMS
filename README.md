# EMS — веб-приложение для CSV (freq / ampl)

Репозиторий: [github.com/nickitabogdanow/EMS](https://github.com/nickitabogdanow/EMS)

```bash
git clone https://github.com/nickitabogdanow/EMS.git
cd EMS
```

Веб-интерфейс на **FastAPI** для загрузки спектров в формате CSV (колонки `freq`, `ampl`): **вычитание** двух кривых, **объединение** по частоте, интерактивные **графики Plotly**, выгрузка результата и HTML.

## Возможности

- Загрузка **нескольких CSV** в браузере (drag-and-drop или выбор файлов), выбор пары файлов для операции.
- **Вычитание:** A − B или B − A по совпадающим значениям `freq`.
- **Объединение:** объединение частот из обоих файлов с сортировкой; при совпадении `freq` — среднее, значение из A или из B.
- **График:** отображение исходных A/B и результата (чекбоксы); по умолчанию можно оставить только одну кривую.
- **Подсветка при вычитании:** полосы на графике, где |разность| не меньше порога; **зелёный** — разность &gt; 0, **красный** — &lt; 0.
- Скачивание **result.csv** / **merged.csv** и **chart.html** (автономная страница с графиком).
- Темы оформления: тёмная, светлая, Barbie, «Единороги» (фон SVG).
- Запуск через **Docker** или локальный **Python**.

## Требования

- Python **3.12+** (рекомендуется) или образ Docker с Python 3.12.

## Быстрый старт (локально)

```bash
cd EMS   # или каталог, куда вы склонировали проект
python3 -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

Запуск (из корня проекта):

```bash
./ems          # macOS / Linux
python run.py  # или явно
```

Либо:

```bash
uvicorn app.main:app --host 127.0.0.1 --port 8000
```

Откройте в браузере: **http://127.0.0.1:8000/**

> Страницу нужно открывать **через HTTP**, а не как файл `index.html` с диска — иначе запросы к API не работают.

### Windows

```bat
ems.bat
```

## Docker

```bash
docker compose up --build
```

Приложение: **http://localhost:8000/**

Чтобы смонтировать свою папку с CSV, раскомментируйте `volumes` в `docker-compose.yml`.

## Формат CSV

- Разделитель: запятая или точка с запятой.
- Заголовок с колонками **`freq`** и **`ampl`** (допускаются варианты `frequency`, `amplitude`, `f`, `a` и т.п.).
- Кодировка: **UTF-8**.

## API (кратко)

| Метод | Путь | Описание |
|--------|------|----------|
| GET | `/` | Главная страница |
| POST | `/api/analyze` | Вычитание: `file_a`, `file_b`, `operation` (`a_minus_b` / `b_minus_a`), флаги отображения, `highlight_threshold` (дБ, 0 = без подсветки) |
| POST | `/api/merge` | Объединение: `duplicate_policy` (`average` / `a` / `b`), файлы и флаги графика |

Файлы в запросе **не сохраняются на диск** сервера: обработка в памяти на время запроса.
