Файлы в сессии

Перетащите сюда несколько CSV или нажмите для выбора. Повторный выбор добавляет файлы к списку, не заменяет его.

Несколько файлов в окне выбора: macOS — удерживайте Cmd и щёлкайте по файлам (или Shift для диапазона). Windows / Linux — Ctrl + щелчок. Без этого система обычно оставляет только один файл.
